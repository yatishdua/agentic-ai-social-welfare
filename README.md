# AI-Driven Social Support Application Automation

This project implements an end-to-end **agentic AI workflow** to automate social support application processing for a government welfare department.

## Key Capabilities
- Multimodal data ingestion (PDF, images, Excel)
- Hybrid ML + Policy-based eligibility assessment
- Disability-aware eligibility prioritization
- Bias & fairness monitoring
- Explainable AI decisions
- Human-in-the-loop governance
- Agentic orchestration using GenAI

## Tech Stack
- Python
- Scikit-learn
- LangGraph
- Ollama / OpenAI (LLM)
- PostgreSQL
- ChromaDB
- Streamlit
- FastAPI
